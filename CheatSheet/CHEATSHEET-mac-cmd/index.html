MAC命令行快捷键

Dec 22, 2020 | CheatSheet
CheatSheet DESC
CTRL + a 左移到行首
CTRL + e 右移到行尾
CTRL + b 左移一格
CTRL + f 右移一格
ESC + b 左移一词
ESC + f 右移一词
CTRL + d 删除当前字符
CTRL + h 删除左边一个字符
CTRL + w 删除左边一个词 (不包含当前字符)
CTRL + k 删除当前及右边所有字符
ESC + DEL 清除至行首 (不包含当前及右边所有字符)
CTRL + u 清除当前行
CTRL + t 交换光标当前字符和左边一个字符的位置
ESC + t 交换光标当前词和左边一词的位置
(当前空格右边如果有词，那空格算右边一词的，否则算前面一词的)
CTRL + c 杀死当前进程
CTRL + z 挂起当前进程 (fg恢复)
CTRL + l 清屏
CTRL + r 搜索历史命令
CTRL + y 粘贴上次的删除
