Git Commit Message Conventions

Aug 14, 2020 | CheatSheet
Contents
  1. 1. 格式
    1. 1.1. type (必须)
    2. 1.2. scope (可选)
    3. 1.3. subject (必须)
  2. 2. Reference 参考

格式

type (必须)

  • feat: 新功能, feature
  • fix: bugfix
  • docs: 文档
  • style: 代码格式化
  • refacotr: 重构
  • perf: 优化
  • test: 增加测试
  • chore: 维护
  • revert: 回滚
  • merge: 代码合并
  • sync: 同步主线或者分支

scope (可选)

scope用于说明 commit 影响的范围

subject (必须)

subject是 commit 目的的简短描述
