格式

type (必须)

  • feat: 新功能, feature
  • fix: bugfix
  • docs: 文档
  • style: 代码格式化
  • refacotr: 重构
  • perf: 优化
  • test: 增加测试
  • chore: 维护
  • revert: 回滚
  • merge: 代码合并
  • sync: 同步主线或者分支

scope (可选)

scope用于说明 commit 影响的范围

subject (必须)

subject是 commit 目的的简短描述
